Incident #104 - Missing Patient Invoices

HIGH SEVERITY HEALTHCARE ERP

Billing process failure detected in production system

1. Issue Description

Hospital reports 53 completed visits without generated invoices.

2. Data Investigation

SELECT patient_id, visit_date FROM visits WHERE invoice_id IS NULL;

→ 53 records found with missing billing reference.

3. System Check

SELECT job_id, status FROM billing_jobs WHERE status = 'FAILED';

→ Billing job failed after last deployment update.

4. Root Cause

Timezone misalignment caused batch processing failure in billing pipeline.

5. Resolution

  • Fixed timezone configuration
  • Reprocessed billing batch
  • Validated invoice generation
