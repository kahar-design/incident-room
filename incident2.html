Incident #087 - Negative Warehouse Stock

CRITICAL SUPPLY CHAIN SYSTEM

Inventory inconsistency detected across warehouse database

1. Issue Description

Multiple products showing negative stock after shipment processing.

2. Data Check

SELECT product_id, quantity FROM inventory WHERE quantity < 0;

→ 17 SKUs affected by negative stock values.

3. Shipment Analysis

SELECT shipment_id, status FROM shipments WHERE processed_before_receipt = 1;

→ Shipments processed before goods receipt confirmation.

4. Root Cause

Process misalignment between warehouse receiving and shipment posting.

5. Resolution

  • Corrected process workflow
  • Rebalanced inventory tables
  • Added validation rule before shipment posting
